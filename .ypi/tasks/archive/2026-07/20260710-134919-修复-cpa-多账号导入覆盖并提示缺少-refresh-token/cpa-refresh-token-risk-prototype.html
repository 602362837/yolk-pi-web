添加 OpenAI Codex 账号
选择一种账号添加方式。
请粘贴由 CPA 转换得到的 credential 数组或对象。必填字段为 type, access, refresh 和 expires。 多账号支持：即使真实 chatgpt_account_id 相同，导入的每条凭证都会被分配独立的存储标识 (storageId) 并独立保存，绝不覆盖已有凭据。
CPA 输入格式示例：
{
  "type": "codex",
  "email": "user@example.com",
  "account_id": "acc_12345",
  "access_token": "eyJ...",
  "refresh_token": "...",
  "expired": "2026-06-08T10:00:00Z"
}
无 Refresh Token 说明：
部分 CPA 凭证可能由于授权方式限制只包含 access_token。虽然过期后无法自动刷新，但 yolk pi web 允许在 access token 有效期内进行导入和临时使用。
