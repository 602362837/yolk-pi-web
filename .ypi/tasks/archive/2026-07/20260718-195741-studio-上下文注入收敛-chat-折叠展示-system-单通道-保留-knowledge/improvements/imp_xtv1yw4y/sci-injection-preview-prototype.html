IMP-001 · Studio tag 注入预览

SCI L0 干净气泡保留；仅在历史脏消息的 compact tag 上提供点击 → 只读 popover，查看被剥离的注入块并复制。

能力边界：预览 = 该条 user 消息里历史写入、现被 strip 的 <ypi-studio-*> 块。 不是当前 agent 本轮 system 通道 live 注入。 L1 之后的新用户消息通常无 tag、无可预览 user 注入。
Esc / 点击外侧关闭 · 键盘可聚焦 tag
