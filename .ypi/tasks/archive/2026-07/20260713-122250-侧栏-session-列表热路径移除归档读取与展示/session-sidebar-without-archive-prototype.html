侧栏 session 列表热路径移除归档读取与展示 — 原型交互

本页面为 UI designer 交付的 HTML 交互原型。它完整体现了在侧栏中剥离“已归档”折叠行及状态以提升热路径性能后的最终视觉体验与交互。

验证检查点：
  • ✅ 移除归档区块： 侧栏会话列表下方彻底没有“已归档”列表和折叠组件。
  • ✅ 批量与单个归档动作： 选中/hover 会话并进行归档，该活动会话将从侧栏消失（在原型中点击可体验）。
  • ✅ 归档确认数量： 点击“归档所有会话”确认时，提示文案的数量口径仅统计 active 会话（不计已在存储中的归档 session）。
  • ✅ 空态自适应： 切换为“空 active 列表”状态时，直接显示“No sessions found in this space”，而不显示任何已归档折叠展开组件。
