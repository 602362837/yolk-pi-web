状态模拟器

  • 1. Planning (无原型) planning
    UI gate required，等待设计员完成 run 与 HTML。
  • 2. Planning (原型已出) planning
    ui-designer 运行成功，原型就绪可请求审批。
  • 3. Awaiting Approval (R1) awaiting
    R1 待用户评审，含 6 个 artifact 链接与 HTML 预览。
  • 4. 计划变动 (Revise) awaiting
    用户修改计划，原 grant 失效，升至 R2 重新审批。
  • 5. 验收开始 (无子任务) acceptance
    主任务进入 user_acceptance，用户可创建改进子任务。
  • 6. 验收阻塞 (子任务处理中) blocked
    有未完成的改进子任务，主任务状态标为等待并阻塞。
  • 7. 改进完成 (回 review) review
    所有改进完成，主任务自动回到 review，需重新验收。
