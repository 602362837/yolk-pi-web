高级：环境变量覆盖

仅用于 CI、容器和专业部署。非空 env 按字段覆盖本机值；普通本机用户无需设置。

env > 本机
i环境变量不会写回本机凭据。若只覆盖部分字段，请确保 env 与本机 fallback 属于同一个 GitHub App。
YPI_GITHUB_APP_ID覆盖 App ID
YPI_GITHUB_APP_PRIVATE_KEY_FILE覆盖私钥，值为部署环境中的 PEM 路径
YPI_GITHUB_APP_WEBHOOK_SECRET覆盖 Webhook HMAC secret
YPI_GITHUB_APP_SLUG可选展示 slug

移除本机凭据？

只删除本机保存的 GitHub App 凭据。不会删除 GitHub App、installation、允许仓库、jobs，也不会修改环境变量。若 env 仍存在，当前进程可能继续显示已配置。
