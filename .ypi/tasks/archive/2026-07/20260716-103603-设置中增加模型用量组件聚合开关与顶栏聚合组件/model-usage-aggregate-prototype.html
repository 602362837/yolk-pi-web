UI v6 规范设计说明 (v6 UI Design Specs)

  • 统一多环与层身份 (Multi-ring & Layer Identity)：
    • 1 个安全窗口 = 1 圈环；N 个安全且可靠排序的窗口 = N 个同心环（外长内短）。
    • 每层具有稳定且明显可区分的 **层身份/笔触风格** (不再仅靠透明度) —— 最外层：Solid 实线； 第二层/中层：Dashed 长虚线； 第三层/内层：Dotted 短虚线。
    • 即使多层同时进入 warning (黄) 或 danger (红) 警戒色，仍然可以借由清晰的笔触风格区分不同的周期窗口。
  • 中心规则与 Unknown 降级 (Center Value & Unknown Fallback)：
    • 中心值一律锁定 **最内层可用层 (Innermost Layer)** 周期标签与百分比。
    • GPT 默认为 5h，Grok 默认为 **周 (Week)**，Kiro 根据多窗口排序。
    • 如果最内层 percent 为 null (未知)，中心百分比显示 `—`，绝不借用外层的百分比数据误导用户。
  • CSS 流光动画 (Subtle CSS-only Used Arc Flow)：
    • 通过 SVG `clipPath` 将流光高光图层精确限制在 **已使用弧段 (Used arc)** 内部移动，完全不溢出至灰色的 empty track。
    • 流光是 CSS 驱动的，无任何 JS Timer 或 React state 刷新。0% 或 null 状态无流光。
    • 完美适配 `prefers-reduced-motion`，当开启减弱动画时，流光高光完全静止/隐藏。
  • 聚合交互与生命周期 (Hover/Focus Life Cycle & Grace Timer)：
    • 聚合面板不再是 click-primary，而是通过 pointer hover 或 keyboard focus (Tab) 触发打开。
    • 220ms 延时关闭 grace window：指针或焦点离开 Trigger 与 Panel 组成的整体交互区时，启动 220ms 延时，在间隙或快速扫过时防止面板闪退；重新进入两者任何区域会取消关闭。
    • Escape 防重开抑制 (Suppression)：按 Escape 键关闭面板并还焦给 Trigger；同时设置临时 suppression，防止 focus 重置事件立即导致面板再次被唤起，直到 Trigger 失去焦点或指针重新移入。
  • 非手风琴分栏面板 (Non-Accordion Columns Grid)：
    • 取消了旧版的 Accordion (折叠手风琴) 一次仅看一家的设定。所有的启用提供商 (GPT / Grok / Kiro) column 同时平铺于 DOM 中展示。
    • Desktop 自动根据启用数渲染为 1–3 列 Grid；在 640px/375px/320px 窄视口下智能响应为 2列/1列，以纵向卡片平铺，绝不隐藏任何服务商。
