- git diff / 新增文件；
- 目标函数是否存在；
- 关键行为测试是否存在并通过；
- UI/API 实际请求是否改变。

### 流程根因
IMP-001 的机器可读 implementation plan 在用户审批后才补录，导致 Studio 初始无法调度；后续状态同步恢复时又把运行成功误当成实施完成。
