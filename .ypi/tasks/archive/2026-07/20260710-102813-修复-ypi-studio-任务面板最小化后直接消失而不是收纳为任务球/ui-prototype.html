UI 门禁验证点提示：
1. 卡片本体点击分责：点击卡片空白处或文本进入卡片阅读态，不会打开右侧 Drawer；只有点击右上角毛玻璃 Detail 圆形按钮（含有指向右的箭头）才会拉出右侧 Drawer 并聚焦任务。
2. 站点连线式进度：卡片中间直接渲染 Workflow 各个节点的拓扑连线，当前进行的步骤会亮起（例如 UI 设计阶段、实现阶段等），进度一目了然。
3. 收纳球保护：在展开面板点 “➖” 收纳。当右侧抽屉打开并聚焦本任务时，球/面板仅变淡避让，禁止消失！
