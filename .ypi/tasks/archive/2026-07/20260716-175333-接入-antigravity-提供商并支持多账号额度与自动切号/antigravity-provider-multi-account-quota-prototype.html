⚠️ 安全与非官方通道提示 (Antigravity Scope)
本组件使用非官方 Google Cloud Code 通道进行模型调用与额度刷新。OAuth 授权将获得 cloud-platform (GCP 完整资源读写) 等宽 scope 权限。请确保已知悉非官方 SLA 限制与官方 IDE 客户端模拟风险。本系统绝对不会收集或上报您的 client_secret、token 凭证或 projectId。

ℹ️ Fail-closed 降级与安全设计说明
自动切号对模型感知 (Model-aware)。候选账号必须具有当前请求模型的 fresh 且 remainingFraction > 0 额度；若仅其他模型有额度、或对应配额状态过期 (stale) / 无法识别 (unknown)，切号模块将安全阻断 (fail-closed) 以防产生盲切或 token 损耗。
